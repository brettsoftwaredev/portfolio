Frogger

In this project I was given a broken game engine and needed to create a working frogger game. This project was mostly focused on useing Javascript. I implement a level based system that gets harder after succesfully completeing a level and reset back to level 1 if you lose. It also keeps track of your highest level achieved.

Matching Game

This project was was mainly focused on Javascript, but I also used some HTML, CSS, and an external library to compliment my project. I used Javascript to create the functions to match, or reset the cards depending if they matched or not. I also created a counter that will track your moves which works with the stars to show how well you did. Upon completion of the game I used an external library call Sweet Alert that pops up and congratulates the player on winning.

Restaurant Reviews
